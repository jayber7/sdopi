# CAO Gestión

Sistema de gestión de Certificados de Avance de Obra para control físico-financiero de proyectos viales.

## Stack

- **Backend:** NestJS + Prisma + SQLite
- **Frontend:** Next.js + React + Tailwind CSS
- **Lenguaje:** TypeScript (estricto)

## Requisitos

- Node.js 20+
- npm 10+

## Desarrollo

```bash
npm install
npm run db:generate
npm run db:push
npm run db:seed
npm run dev
```

Backend: http://localhost:3001
Frontend: http://localhost:3000
